# 🚀 Premium Fintech Landing

A modern, high-performance fintech landing page built with **Next.js 15**, **TypeScript**, **Tailwind CSS**, and **Framer Motion**.

Designed with a strong focus on premium UI, smooth interactions, responsive layouts, accessibility, and production-ready architecture.

> **Disclaimer:** This is an independent educational and portfolio project inspired by modern fintech design systems. It is not affiliated with or endorsed by any existing fintech company.

---

## ✨ Features

* Modern premium fintech UI
* Responsive across desktop, tablet, and mobile
* Smooth Framer Motion animations
* Glassmorphism design system
* Reusable component architecture
* Optimized typography and spacing
* Interactive FAQ
* Animated statistics
* Premium CTA sections
* Production-ready Next.js App Router
* TypeScript support

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/PawanChoudhary0607/Premium-fintech-landing.git
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Start production server

```bash
npm run start
```
